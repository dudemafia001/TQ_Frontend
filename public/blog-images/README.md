# Blog Images Folder

Upload your blog featured images here.

## How to use:

1. **Upload your image** to this folder (e.g., `my-blog-image.jpg`)
2. **In the admin dashboard**, use the path: `/blog-images/my-blog-image.jpg`

## Supported formats:
- JPG/JPEG
- PNG
- WebP
- GIF

## Recommendations:
- **Image size**: Optimize images to under 500KB for faster loading
- **Dimensions**: Recommended 1200x630px (16:9 ratio) for best display
- **File naming**: Use lowercase with hyphens (e.g., `bulk-food-delivery.jpg`)

## Example:
If you upload: `bulk-food-kanpur.jpg`  
Use in admin: `/blog-images/bulk-food-kanpur.jpg`
